# Demographic-Income-Prediction-Logistic-Regression-vs.-KNN-Analysis
 Income Predictor: An ML pipeline classifying census data into income brackets. Features automated cleaning, EDA, and an OOP-based comparison of Logistic Regression vs. KNN. Includes hyperparameter tuning (K=16) achieving 84.36% accuracy. Built with Scikit-Learn and Seaborn for professional demographic analysis.
